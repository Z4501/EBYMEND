# Statement-Driven eBay P&L Viewer

Upload:
- Statement Excel (.xlsx)
- eBay Orders CSV
- ShipStation CSV

The statement drives which orders appear.

Matching key:
- Statement `Cust. P.O.`
- eBay `Order Number`
- ShipStation `Order #`

This version is much more accurate than PDF parsing because it reads statement rows directly from Excel.
